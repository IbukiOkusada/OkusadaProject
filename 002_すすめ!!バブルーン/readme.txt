//==================================================================
// 説明
//==================================================================
serverフォルダに通信用サーバープロジェクト、gameフォルダにゲーム本編が入っています。
書類フォルダには作成したワンシート企画書とガントチャートが入っています。
発表会では2台のPCを用いたため一台でのPCではテストに不便かもしれないので動画をご視聴ください。
ビルドする際は設定をx64からx86に変更をお願いします(サーバーの方は変更しなくて問題ありません)。
動作環境 Windows11にて確認済みです。

//==================================================================
// ゲームのルール
//==================================================================
障害物を避けながら制限時間内にゴールを目指す
対戦モードではどちらが先にゴールするかを二人プレイで遊べます。

//==================================================================
// アピールポイント
//==================================================================
TCPとwinsockを用いてオンラインの通信を実装しました。同期させる部分が不安定ですが、
スタートしてゴールするまで座標、向き、体力などを送受信しています。
接続先サーバーのアドレスはdata/TXT/address.txtに入力して接続可能です。

モデルの形の影を実装しています。シャドウマトリックスを用いて地面に描画しています。
描画するかどうかを設定して丸影とも組み合わせています。

//==================================================================
// 各バージョン進捗
//==================================================================
2023/10/27にマスター版で2023/10/02から製作開始。
シンマスター版は学内発表会前のブラッシュアップでありました。

α版 2023/10/13
プレイヤーの操作、加速と慣性で動く形にしています。
回転オブジェクトのと対角線とベクトルを使用した当たり判定と動く物との反射。

β版 2023/10/20
プレイヤーがシャボン玉で距離を取った当たり判定を設けています。

マスター版 2023/10/27
TCPとwinsockを使用したオンラインの通信を実装。開始からゴールまで実装。
勝敗をリザルトに反映。勝ち負けも分かれるようになっています。

シンマスター版 2023/10/30
オンラインの反映を調整、UI周りのテクスチャを一部世界観に合わせる。ミニマップの追加。
開始時の目標、ゴール通知を追加
