// タイトル Motion Viewer
//==================================================================

//==================================================================
// 説明
//==================================================================
exeフォルダに実行ファイルが入っています。
projectフォルダに開発環境が入っています。
ビルドする際は設定をx64からx86に変更をお願いします。
動作環境 Windows11にて確認済みです。

//==================================================================
// 概要
//==================================================================
実際のゲームで使用した二段の階層構造のモーションを
作成して外部ファイルに書き出してそのままゲームに使える
ツールです。キー数やフレーム数なども自由に変更が可能です。

//==================================================================
// アピールポイント
//==================================================================
上半身と下半身別々にモーションを作成することができて
ツール上で動作確認と作成が可能になっています。 
データは全て外部ファイルへ書き出して保存。
読み込んだファイルを書き換えて
そのままゲームに実装できるようになっています。
