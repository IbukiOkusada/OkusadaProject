動作環境 Windows11にて確認済みです。

//==================================================================
// ゲームのルール
//==================================================================
制限時間中、市民が熱中症で倒れないようにプレイヤーを
動かして水鉄砲を当てて冷やし続けるゲームです。
時間ごとに市民が出たり入ったりするので現在の出現位置と
固まり方を見て見極める楽しさがあります。

//==================================================================
// アピールポイント
//==================================================================
全体の一回で入る値をすべて管理してゆっくりに見えるようなスロー、
逆に加速させて定点カメラの倍速映像のような表現をしています。
またプレイヤーを二段の階層構造にすることで上下別の動きをしてより
アクションを見せられるように、モーション管理をやりやすくしています。
